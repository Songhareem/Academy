
# 다형성

- 인스턴스의 취급이 변수의 형태에 의해 변하는것
- 부모 타입으로 자식 클래스를 참조
    - 상속시켜준 멤버만 접근 가능
    - Parent P = new Child();
        - 왼쪽 객체가 오른 쪽 객체의 상위 클래인 경우 자동 형변환
    - 기본형 데이터들처럼 참조 자료형 변수들도 형변환 

- upcasting
    - 자식을 부모 클래스의 부모 타입으로 형변환
    - 묵시적으로 형변환 가능

- downcasting
    - 부모를 자식 타입으로 형변환
    - 명시적 형변환 필요
